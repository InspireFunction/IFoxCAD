# 关于扩展函数的说明

## 一、命名规则

扩展函数全部都放在 **ExtensionMethod** 文件夹里。

然后每个类别都需要分别建立一个以EX为结尾的文件。比如：

DBObjectEx                  实体对象扩展类

SymbolTableRecordEx         实体对象扩展类

EditorEx.cs                 表示关于命令的扩展
